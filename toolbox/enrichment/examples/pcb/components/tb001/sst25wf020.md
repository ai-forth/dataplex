## Summary

`SST25WF020` is the BOM-backed SPI flash identity for `U3` on TB001 page 1. It
is a low-voltage serial flash part that helps constrain the board's boot,
programming, and non-volatile configuration behavior around the GA144 control
region.

## Known From Current Board Evidence

* The part is identified in the BOM as `SST25WF020`.
* It appears as `U3` on page 1.
* The current TB001 notes describe it as SPI flash on the control, memory, and
  flash page.
* The scaffold and workbook associate it with `SOP-8`.

## Manufacturer Characteristic Status

Known:

* MPN string: `SST25WF020`
* manufacturer family: SST / Microchip `SST25WF` serial flash
* device class: SPI serial flash memory
* density: `2 Mbit`
* operating voltage: `1.65 V to 1.95 V`
* interface compatibility: SPI `Mode 0` and `Mode 3`
* maximum clock frequency: `40 MHz`
* endurance: `100,000` cycles
* data retention: greater than `100 years`
* typical active read current: `2 mA` at `20 MHz`
* typical standby current: `2 uA`
* erase granularity:
  * uniform `4 KByte` sectors
  * uniform `32 KByte` overlay blocks
  * uniform `64 KByte` overlay blocks for larger family members
* package family:
  * `8-lead SOIC`
  * `8-contact WSON`

Still required from manufacturer sources:

* normalized statement of which exact package variant TB001 uses
* explicit board-level capture of boot/reset/write-protect wiring intent
* later review of whether the `RST#/HOLD#` option is used as reset or hold on
  TB001

## Manufacturer Source Intake

The local primary-source datasheet at `../TB001/datasheets/SST25WF020.pdf`
identifies `SST25WF020` as part of the low-voltage `SST25WF` SPI flash family.

The same sheet also resolves the earlier filename ambiguity:

* `DS20005016C` is the datasheet document number
* it is not a TB001 part number

Relevant pin structure from the datasheet:

* `CE#`
* `SO`
* `WP#`
* `VSS`
* `SI`
* `SCK`
* `RST#/HOLD#`
* `VDD`

These align well with the existing workbook notes for `U3`.
